# simple-logger-usage Skill

Expert guide for integrating @alcyone-labs/simple-logger into TypeScript applications.

## What This Skill Covers

- File-scoped logger patterns for backend, frontend, and shared libraries
- Chrome MV3 service worker compatibility (IIFE bundle)
- Transport configuration (Console, Remote, PinoBridge)
- Request-context and child logger chaining
- Best practices for metadata and data separation

## Quick Start

1. Load the skill
2. Request logging for a specific file or component
3. Get context-aware logger implementation

## Topics

- **usage**: Core logging patterns, API reference, configuration, and gotchas

## Commands

- OpenCode: `/simple-logger-usage [task]`
- Gemini CLI: `simple-logger-usage [task]`
- FactoryAI Droid: `simple-logger-usage [task]`

## Install

```bash
# Global
curl -fsSL https://raw.githubusercontent.com/Alcyone-Labs/simple-logger/main/skills/install.sh | bash

# Local
./skills/install.sh --local
```
